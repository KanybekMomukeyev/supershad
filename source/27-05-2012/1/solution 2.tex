Выбираем сначала $n$ шагов по одной из осей из $6n$ возможных: $C_{6n}^n$. Выбираем $2n$ шагов из оставшихся $5n$ шагов: $C_{5n}^{2n}$. Общее число комбинаций равно $C_{6n}^n C_{5n}^{2n} = \frac{(6n)!}{n! (2n)! (3n)!}$.
